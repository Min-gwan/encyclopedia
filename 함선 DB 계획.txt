1~2차 세계대전 함선 DB 만들기

필요한 테이블

1. 함종 분류 코드 테이블
- ID(함종 분류 코드), TYPE,

2. 국가 코드 테이블
- ID(국가 코드), NATION(국가명)

3. 시기 구분 코드
- W_CODE(세계대전 구분), DESCRIPTION

4. 국가별 함종별 실존함 테이블
- N_CODE(국가코드), S_CODE(함종 코드), NAME(함선명),
